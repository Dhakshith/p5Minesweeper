p5Minesweeper
===
*A Minesweeper Game Using p5.js.*
---
You Can Change The Width Of Blocks By Changing
<br>
<br>
`var w = 50;`
<br>
<br>
Into
<br>
<br>
`var w = ` Width `;`
---
You Can Also Change The Total Number Of Bees By Changing
<br>
<br>
`var totalBees = 60;`
<br>
<br>
Into
<br>
<br>
`var totalBees = ` Total Number Of Bees `;`
